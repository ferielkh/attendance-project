# Attendance Management System

**⚠️ Note: The final project files are in the master branch of the repository.**

This project is an Attendance Management System built for the PAW course. It allows you to:

- Add new students
- Track attendance and participation for multiple sessions
- Highlight excellent students
- Generate a simple attendance report with charts

## Technologies Used
- HTML
- CSS
- JavaScript (with jQuery and Chart.js)
- PHP (for form validation)
